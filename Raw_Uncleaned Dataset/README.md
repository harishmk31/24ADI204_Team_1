## Description
This file contains the original raw IPL 2025 Auction dataset obtained from the source.
The dataset includes player information such as name, country, age, role, caps,
IPL matches, reserve price, and other attributes.

This dataset has not been modified and may contain:
- Missing values
- Inconsistent formatting
- Swapped or misplaced values
- Text-based price formats (Cr/Lakh)
- Duplicate records

## Data Characteristics
- Contains raw auction registration details
- Includes both Indian and international players
- Reserve price stored in text format (e.g., "2 Cr", "50 Lakh")
- Some columns may contain missing or incomplete values

## Purpose of Raw Dataset
The raw dataset is preserved to:
- Maintain data integrity
- Ensure reproducibility of cleaning steps
- Compare before-and-after preprocessing results
- Track all data transformation decisions

## Important Note
This file should not be used directly for analysis.
Please use the cleaned dataset (cricket_cleaned.csv) for
EDA, statistical analysis, and modeling tasks.
